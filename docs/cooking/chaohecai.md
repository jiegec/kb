# 炒合菜

## 食材

豆芽 鸡蛋 韭菜 绿豆芽 红薯粉条

## 步骤

1. 热水泡红薯粉条，泡软
2. 打好蛋液，在锅中炒鸡蛋，边搅拌，边倒入蛋液，盛出
3. 翻炒豆芽，加入粉条，调味
4. 最后倒入韭菜和炒好的鸡蛋继续翻炒
