# 菠菜炒鸡蛋

## 食材

菠菜 鸡蛋

## 步骤

1. 菠菜洗净 盐水泡一段时间
2. 可选菠菜切根
3. 锅中把水烧开，下盐和油，下菠菜，焯水到段生
4. 把菠菜取出放到冷水，过冷河
5. 用厨房纸吸去菠菜上的水，切成 2cm 的段
6. 把菠菜放到打好的蛋液中搅拌；也可以不搅拌，把鸡蛋炒好以后和菠菜混合
7. 热锅冷油，把菠菜和蛋液下锅
