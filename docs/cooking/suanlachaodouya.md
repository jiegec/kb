# 酸辣炒豆芽

## 食材

绿豆芽

## 步骤

1. 清洗豆芽，把水沥干
2. 下锅先把辣椒等调味料爆香，注意辣椒不要放太多，导致辣味压过酸味
3. 豆芽下锅，尽量把豆芽平铺在锅中，持续翻炒，火可以小一些
4. 下盐，生抽，醋（白醋 + 陈醋），持续翻炒，不出汤水比较好，不用炒太透

## 记录

- 2025-03-27: 做了一次，300g 绿豆芽加一根小米辣，辣味太大了，应该只需要半根，此外醋可以多放一些
