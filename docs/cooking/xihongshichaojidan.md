# 西红柿炒鸡蛋

## 食材

西红柿 鸡蛋 番茄酱

## 步骤

1. 清洗西红柿，可选剥皮，切成小块
2. 煎鸡蛋，可以参考 [煎蛋](./jiandan.md) 的方法，根据口味选择做法
3. 煎好鸡蛋后把鸡蛋取出，切成小片
4. 下切好的西红柿翻炒，后再加入鸡蛋，加番茄酱调味

## 记录

- 2025-04-15: 做了一次，200g 鸡蛋加 300g 个头比较小的水果番茄，不好剥皮，偷懒没有按照视频里的方法，切开皮再热水煮再撕皮，不过带着一点皮口感也还行，可能和品种有关；加了番茄酱，味道更好一些
