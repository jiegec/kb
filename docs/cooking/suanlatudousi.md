# 酸辣土豆丝

## 食材

土豆

## 步骤

1. 土豆切成 3mm 左右厚的细丝，可以用擦丝器，重复洗多次，把淀粉洗掉
2. 下锅先把辣椒等调味料爆香，注意辣椒不要放太多，导致辣味压过酸味
3. 土豆下锅，持续翻炒，下盐，生抽，醋（白醋 + 陈醋），持续翻炒

## 记录

- 2025-03-28: 做了一次，500g 土豆加半根小米辣，辣味淡了点，此外酸味有点不足，可能是陈醋放的不够，可以考虑加酸味比较浓的白醋
