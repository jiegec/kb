# 炒芹菜

## 食材

芹菜

## 步骤

1. 去掉叶子和根，撕掉芹菜的筋，切菱形块，不要太厚
2. 把芹菜下到开水，下盐和油，焯水，捞出过冷河
3. 下油把蒜爆香，加少量水和盐，下芹菜，翻炒
