# 葱爆肉

## 食材

大葱 猪肉或羊肉或鸡肉

## 步骤

1. 大葱切段或滚刀块，肉切片
2. 肉切片，洗干净，挤干水后调味，可选加蛋清，加淀粉，搅拌，可选加油
3. 肉片下锅，滑炒，倒出
4. 炒大葱，调味，倒入肉片
