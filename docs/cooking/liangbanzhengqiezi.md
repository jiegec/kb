# 凉拌蒸茄子

## 食材

茄子 葱 蒜

## 步骤

1. 把茄子纵向切开，蒸到比较软的状态，算上烧开水时间大概十多分钟，可以用电饭煲蒸，如果电饭煲不够大，可以把茄子切短一点
2. 调一个料汁：葱剁碎，拍蒜蓉，加陈醋，生抽，盐，糖，这里面味道主要是醋和蒜
3. 茄子蒸好以后撕成小条，放凉一点，浇上料汁，拌匀即可
